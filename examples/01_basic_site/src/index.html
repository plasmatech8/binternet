This site was inscribed onto the Bitcoin Blockchain!

The HTML/CSS/JS and image files were inscribed onto the blockchain.

After these files were inscribed onto the blockchain, a router inscription (defined as a YAML file) was created which maps URL paths to the website assets via inscription number.

For example:


    binternet: v1
    routes:
        /: 1001             # /index.html
        /styles.css: 1002   # /styles.css
        /script.js: 1003    # /script.js
                            # ...etc
